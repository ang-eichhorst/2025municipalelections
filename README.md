# 2025municipalelections
How we tracked CT's 2025 municipal elections

State website publishes unofficial election results by town  https://ctemspublic.tgstg.net/#/selectTown 

Get a Google Cloud account for writing into Google Sheets

Remember to edit election year (can find when you inspect page source, ie 2025 was 97), and make sure google sheet file names match. 

Run angela_script in terminal "source angela_script.sh"
